# Protobuf 聊天客户端完整使用说明

## 编译和运行

### 1. 编译项目
```bash
# Windows
cd j:\集群聊天服务器\Server-main\Service
build.bat

# Linux
cd /path/to/Service
chmod +x build.sh
./build.sh
```

### 2. 启动服务器
```bash
./bin/ChatServer 127.0.0.1 8000
```

### 3. 启动客户端
```bash
./bin/ProtobufChatClient 127.0.0.1 8000
```

### 💬 **聊天功能**

#### **一对一聊天**
**命令格式：** `chat:好友ID:消息内容`

```bash
>> chat:1002:你好！
消息已发送给好友 1002

>> chat:1002:今天天气真不错
消息已发送给好友 1002
```

**接收消息显示：**
```
[2025-08-09 14:30:25] Bob(1002) says: 你好，Alice！
[2025-08-09 14:30:30] Bob(1002) says: 我也觉得天气不错
```

#### **群聊**
**命令格式：** `groupchat:群ID:消息内容`

```bash
>> groupchat:101:大家好！
群聊消息已发送到群 101

>> groupchat:101:今天讨论什么技术话题？
群聊消息已发送到群 101
```

**群消息接收显示：**
```
[2025-08-09 15:00:10] Group[101] Charlie(1003) says: 大家下午好！
[2025-08-09 15:00:25] Group[101] David(1004) says: 我们聊聊Go语言吧
```

## 🔧 **技术特点**

- **实时通信**：基于 TCP 长连接，消息实时推送
- **Protobuf 序列化**：高效的二进制协议，性能优于 JSON
- **多线程设计**：发送和接收消息在不同线程，互不阻塞
- **离线消息**：支持离线消息存储，上线时自动接收
- **群组功能**：支持群组创建、加入和群聊
- **好友系统**：支持添加好友和好友聊天

## ⚠️ **注意事项**

1. **好友关系**：添加好友后才能进行一对一聊天
2. **群组权限**：需要先加入群组才能在群里聊天
3. **消息格式**：严格按照命令格式输入，冒号为英文冒号
4. **在线状态**：只有在线用户才能实时收到消息
5. **网络连接**：确保服务器正在运行且网络连接正常
