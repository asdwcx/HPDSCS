# 优化后添加好友协议测试指南

## 🎯 **测试环境准备**

在实际测试之前，需要完成以下准备工作：

### **1. 环境依赖**
```bash
# 安装protobuf编译器和库
sudo apt-get install protobuf-compiler libprotobuf-dev

# 或者在CentOS/RHEL上
sudo yum install protobuf-compiler protobuf-devel

# Windows用户需要下载并安装Protocol Buffers
# https://github.com/protocolbuffers/protobuf/releases
```

### **2. 编译protobuf文件**
```bash
cd j:\集群聊天服务器\Server-main\Service\proto
protoc --cpp_out=../include/ message.proto
```

### **3. 编译项目**
```bash
cd j:\集群聊天服务器\Server-main\Service\build
cmake ..
make
```

## 🧪 **功能测试用例**

### **测试用例1：成功添加好友**

#### **Step 1: 启动服务器**
```bash
cd j:\集群聊天服务器\Server-main\Service\bin
./ChatServer
```

#### **Step 2: 启动客户端A (Alice, ID: 1001)**
```bash
./ChatClient 127.0.0.1 8080

# 登录
login:1001:123456

# 发起添加好友请求
addfriend:1002:我是Alice，想和你成为好友
```

**预期输出：**
```
添加好友请求已发送给用户 1002，等待对方回复...
```

#### **Step 3: 启动客户端B (Bob, ID: 1002)**
```bash
./ChatClient 127.0.0.1 8080

# 登录
login:1002:123456
```

**预期输出：**
```
==================================
[好友请求] Alice(1001) 想要添加你为好友
消息: 我是Alice，想和你成为好友
请回复: accept:1001:是否同意(true/false):回复消息
示例: accept:1001:true:很高兴认识你
==================================
```

#### **Step 4: Bob同意请求**
```bash
accept:1001:true:很高兴认识你，Alice！
```

**Bob端预期输出：**
```
已同意用户 1001 的好友请求
```

**Alice端预期输出：**
```
==================================
[系统通知] 添加好友结果：好友添加成功！
新好友: Bob(1002)
回复: 很高兴认识你，Alice！
现在可以开始聊天了。
==================================
```

## 🔧 **错误情况测试**

### **格式错误测试**

#### **1. 缺少消息参数**
```bash
addfriend:1002
```
**预期输出：**
```
格式错误！正确格式: addfriend:好友ID:请求消息
示例: addfriend:1002:我是Alice，想和你成为好友
```

#### **2. accept命令格式错误**
```bash
accept:1001:maybe:不确定
```
**预期输出：**
```
是否同意参数错误！请输入 true 或 false
```

### **边界条件测试**

#### **1. 添加自己为好友**
```bash
addfriend:1001:想加自己
```
**预期输出：**
```
不能添加自己为好友！
```

#### **2. 无效的用户ID**
```bash
addfriend:0:测试无效ID
```
**预期输出：**
```
好友ID必须是正整数！
```
